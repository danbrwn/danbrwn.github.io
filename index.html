Automated Documenting ControlLogix PLC Functional Descriptions

Developed a series of Node.js scripts to document and validate ControlLogix PLC functional descriptions, translating complex PLC logic into understandable formats.

Challenge: Updating functional descriptions after 15 years of changes across approximately 50 PLCs, and validating settings against best practices, all while making the information comprehensible for users who cannot evaluate PLC rungs directly.

Solution:

  • ControlLogix PLC to Boolean Expressions: Wrote a Node.js script to parse ControlLogix L5K export programs into boolean equivalent statements, using complex regular expressions.
  • Boolean Expression to Natural Language Using OpenAI API: Developed a Node.js script to translate boolean language statements into a natural language synopsis, providing a comprehensive explanation of operations.
  • Document Alarm Values ControlLogix PLC: Created a Node.js script to extract digital and analog alarm block parameters from ControlLogix exported files, simplifying the documentation and validation process for a large set of alarms.

Outcome: The combined efforts allowed for accurate documentation, validation, and explanation of complex PLC code, making it accessible and understandable for end users. This streamlined the process of updating and validating the PLC functional descriptions.

Technology Stack

  • Frontend: MS Excel,VBA
  • Backend: Node.js, OpenAI api

        function extractAnalogAlarmDetails(content) {
            const regex = /(\w+)\s*:\s*(ANALOG|ANALOG_ALARM)\s*\(Description\s*:=\s*"([^"]*)"\)\s*:=\s*\[(.*?)\];/gs;
            const matches = [...content.matchAll(regex)];
            const details = matches.map(match => {
                const tag = match[1];
                const type = match[2];
                let description = match[3].replace(/\$N/g, ' '); // Replace $N with space
                const values = match[4].split(',').map(v => v.trim());
                
                if (type === "ANALOG_ALARM" && values.length >= 16) {
                    return {
                        TAG: tag,
                        Description: description,
                        Hi_Hi: parseFloat(values[12]),
                        Hi: parseFloat(values[13]),
                        Lo_Lo: parseFloat(values[14]),
                        Lo: parseFloat(values[15])
                    };
                } else if (type === "ANALOG" && values.length >= 16) {
                    return {
                        TAG: tag,
                        Description: description,
                        Hi_Hi: parseFloat(values[12]),
                        Hi: parseFloat(values[13]),
                        Lo_Lo: parseFloat(values[14]),
                        Lo: parseFloat(values[15])
                    };
                }
                return null;
            }).filter(detail => detail !== null);
            return details;
        }
